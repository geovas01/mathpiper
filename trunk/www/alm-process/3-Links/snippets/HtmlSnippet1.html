Software used by MathRider:

jEdit
jEdit is a mature programmer's text editor with hundreds (counting the time developing plugins) of person-years of development behind it. MathRider is built on top of jEdit and, if you are looking for a powerful programmer's text editor, jEdit is definitely one to look at.
GeoGebra
GeoGebra is dynamic mathematics software for schools that joins geometry, algebra and calculus.
HotEqn
HotEqn is software for viewing mathematical expressions in traditional form.
JSciCalc
Java Scientific Calculator is a general-purpose scientific calculator that you can use as a computer-desktop calculator or in a web application.
Yacas
Yacas is an easy to use, general purpose computer algebra system for symbolic manipulation of mathematical expressions. Piper (the default computer algebra system which comes with MathRider) is derived from Yacas.

MathRider-related educational materials:

Professor and Pat

Learn how a computer works from the ground up. A hand-held calculator kit which is based on the materials at this site is planned. The idea is that a person would work through the books at this site, assemble their calculator kit, and then program it in 6502 assembly language. Very old school and very exciting!

Java-based mathematics software:

MathEclipse.org

A Java-based Computer Algebra system which has functions for arbitrary-precision integer arithmetic, matrices, vectors, finite sets, derivatives, pattern-matching, term rewriting rules and functional programming.
