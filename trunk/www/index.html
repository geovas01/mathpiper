Welcome to the MathRider project.

 

MathRider is an education-friendly Super Scientific Calculator (SSC) for performing numeric and symbolic computations. 

 

The initial alpha release of MathRider is now available.  It can be downloaded from the "Download" tab shown above.  The support group for MathRider is located at http://groups.google.com/group/mathrider.

 

If you would like to join this project, please select the Observer role :-)
