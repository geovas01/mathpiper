Programming in Piper, function reference (PDF) (download)
This is the second part of the Piper function reference. This reference explains functions that are used in the Piper source code that is underneath what the user sees. The documentation in this section is thus mostly useful to people who are maintaining Piper.

The Piper Book of Algorithms (PDF) (download)
This book is a detailed description of the algorithms used in the Piper system for exact symbolic and arbitrary-precision numerical computations. Very few of these algorithms are new, and most are well-known. The goal of this book is to become a compendium of all relevant issues of design and implementation of these algorithms.

Introduction to the off-line version of Piper (PDF) (download)
This document gives a short introduction to Piper. Included is a brief tutorial on the syntax and some commands to get you started using Piper. There are also some examples.

Essays on Piper (PDF) (download)
This is a book of essays on Piper. It covers various topics, from using the Piper system for specific calculations to general issues related to the Piper system development and maintenance.
