SSU Student Schedule Generator v.002 beta

The purpose of the student schedule generator is to automatically create student schedules. It is a web application that can be accessed with any modern web browser. The current version of the application is configured to use the fall semester 2012 course schedule.

The following screenshot of the application contains labeled arrows that explain the various parts of its user interface:








The application is tab based, and the leftmost tab (which is labeled "Enter Courses") contains the main user interface. These are the steps that are followed in order to generate one or more schedules:

  1. Enter one or more course numbers into the text boxes that are in the "Course Numbers" section.

  2. Select one or more courses in the "Fine Arts" section if desired. If the mouse pointer is "hovered" over any of these course numbers, more information about the course is shown.

  3. The default number of schedules to generate is one. The number of schedules to generate can be increased in the "Number of Schedules" section.

  4. By default, no preferences are selected. A morning, afternoon, or evening preference can be selected in the "Preferences" section if desired.

  5. Press the "Submit" button to generate the number of selected schedules. The schedules are contained in a new tab.
